# Python Basic Exercises Set

Набор базовых упражнений на Python для отработки работы с типами данных, строками, математическими функциями и преобразованиями.

## Примеры

### 1️⃣ Приветствие и подсчет символов
Программа запрашивает фамилию и имя, выводит приветствие и подсчитывает количество символов в каждом слове.

### 2️⃣ Работа с типами и дробными числами
Демонстрация преобразования типов (int/float/str), выделение целой и дробной части числа с заданной точностью.

### 3️⃣ Логический тип
Преобразование строки в булево значение, демонстрация представления True/False как строк и чисел.

### З1️⃣ Операции с цифрами числа
Набор функций для работы с многозначными числами:
- Сумма первой и последней цифры
- Сумма первой и предпоследней цифры
- Сумма первой и второй цифры
- Произведение последней и предпоследней цифры
- Обмен первой и последней цифры
- Обмен первой и предпоследней цифры
- Обмен первой и второй цифры
- Обмен последней и предпоследней цифры
- Создание числа из первых двух цифр одного числа и последних двух другого
- Создание числа из первой цифры первого числа, трех цифр второго и последней цифры первого

### З2️⃣ Математические выражения
Набор функций для вычисления различных математических выражений с использованием модуля math:
- exponent, тригонометрия, логарифмы, корни
- Обработка абсолютных значений
- Комбинированные выражения

## Куда можно встроить

- В учебные материалы по основам Python
- В курс по типам данных и преобразованиям
- Как сборник задач для начинающих
- В математические калькуляторы и обработчики чисел

## Требования

- Python 3.x
- Модуль math (входит в стандартную поставку)

## Примечание

В сборнике представлены задачи разной сложности: от простого ввода/вывода до сложных математических выражений. Особое внимание уделено работе со строками и преобразованию типов.
